# Field-Service-Scheduler



Desktop application for managing and scheduling field technician dispatches.

## Overview

Field Service Scheduler helps operations teams assign service requests to technicians, prevent scheduling conflicts, and track job progress in real time — all from a single calendar view.

## Key Features

- **Calendar / Timeline view** — day and week layouts with per-technician task lanes
- **Smart assignment** — assign a technician to a request with automatic time-conflict detection
- **Request lifecycle** — status flow: `scheduled` → `en route` → `in progress` → `done`
- **Desktop notifications** — Electron-based alerts on every status change
- **Advanced filtering** — filter by technician, service zone, or work type
